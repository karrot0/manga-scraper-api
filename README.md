
# README

## Overview

Manga Scraper API is a scraper that uses external updated extensions and provides a run time.

## Features

> Scraper
> Dashboard
> Database
> API
> MCP

## Installation

```bash
# bun install
```

## Usage

```bash
# bun start
```

## MCP

Start the server and add to your mcp.json

```
{
  "mcpServers": {
    "manga-scraper": {
      "url": "http://localhost:3001/mcp"
    }
  }
}
```
